# 步驟零：市場訊號掃描 Prompt
# 使用時機：每月第一週，在 Claude Code 互動介面執行
# 執行方式：直接貼入 Claude Code，不需要 python 腳本

/last30days semiconductor investment themes trending Reddit HN investor discussion {month}

研究完成後，請用繁體中文回答：

過去 30 天，國際社群（Reddit、HN、X）討論最熱的半導體與科技投資主題前五名是什麼？

每個主題請用表格格式說明：

| 項目 | 內容 |
|------|------|
| 討論熱度 | 高／中／低，並說明來源與互動數 |
| 市場是否已充分定價 | 是／否／不確定，說明理由 |
| 台灣供應鏈受益標的 | 列出 2–3 個可能相關的台股公司 |
| 是否值得本月深入研究 | 是／否，一句話說明原因 |

最後請給出本月建議研究的前三個產業，並說明排除其他主題的邏輯。
